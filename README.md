# Library Management System 📚

This is a Python-based internship project designed to automate the process of a library.

## Key Features:
* **Add Books:** Add new titles to the library database.
* **Issue Books:** Keep track of books issued to students.
* **Return System:** Easily return and update the inventory.
* **Search:** Search for books by title or author.

## Tech Stack:
* Language: Python 3.x
* Database: File Handling (.txt) or SQLite
